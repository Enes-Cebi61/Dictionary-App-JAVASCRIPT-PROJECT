JavaScript Dictionary Application
This project is a simple dictionary application that uses JavaScript to fetch word meanings and pronunciations from an API. Users can type a word into the search bar to learn its meaning and pronunciation. The application uses an API to fetch word definitions and pronunciations, allowing users to search for words.

Features
Word search
Display of word meanings
Text-to-speech pronunciation
Fetching word meanings and pronunciations from an API
Technologies
JavaScript
HTML
CSS
## How to Run

1. Clone this repository.
2. Open the index.html file in a web browser.
